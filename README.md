# Docker-Agent-Testing
Files used in docker agent refactor testing

This Repository Contains docker agent .yaml files sourced from github codesearch used in testing for a school project
To replicate any testing done you'll need to download and run the docker agent application from https://github.com/docker/docker-agent and obtain an openai api key for gpt-4o which you can save as an environment variable.

These files are only being saved to github in case I ever want to reference back to them, so trying to replicate testing without some modification on your own to the agent files and/or creating the missing files they reference won't work.
